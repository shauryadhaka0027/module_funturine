Complete Installation & Setup Guide

üìã Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (version 16.0.0 or higher)
  • npm (version 8.0.0 or higher)
  • MongoDB (local installation or MongoDB Atlas)
  • Git (for cloning the repository)

1Clone the Repository

First, clone the project repository to your local machine:

git clone <repository-url>
cd module_funturine

2Install Dependencies

Install all required dependencies using npm:

npm install

This will install all production and development dependencies listed in package.json.

3Environment Configuration

Create a .env file in the root directory with the following configuration:

# Server Configuration PORT=5000 NODE_ENV=development # Database Configuration MONGODB_URI=mongodb://localhost:27017/moulded-furniture # JWT Configuration JWT_SECRET=your-very-secure-jwt-secret-key-here JWT_EXPIRES_IN=24h # Email Configuration EMAIL_HOST=smtp.gmail.com EMAIL_PORT=587 EMAIL_USER=your-email@gmail.com EMAIL_PASS=your-app-password EMAIL_FROM=noreply@mouldedfurniture.com # Admin Configuration ADMIN_EMAIL=admin@mouldedfurniture.com ADMIN_PASSWORD=admin123
‚ö†Ô∏è Important: Make sure to replace the placeholder values with your actual configuration, especially the JWT_SECRET and email credentials.

4Database Setup

Initialize the database and create the admin user:

npm run init-db
npm run create-admin

Optionally, populate the database with sample products:

npm run populate-products

5Build the Project

Compile the TypeScript code:

npm run build

This will create a dist folder with the compiled JavaScript files.

6Start the Server

For development (with auto-reload):

npm run dev

For production:

npm start
‚úÖ Success! Your server should now be running on http://localhost:5000

7Verify Installation

Test that everything is working correctly:

  • Health check: GET http://localhost:5000/api/health
  • Test email service: npm run test-email
  • Admin login test using the credentials from your .env file

8Available Scripts

Here are all the available npm scripts:

  • npm run dev - Start development server with auto-reload
  • npm start - Start production server
  • npm run build - Build the project
  • npm run lint - Run ESLint
  • npm test - Run tests
  • npm run init-db - Initialize database
  • npm run create-admin - Create admin user
  • npm run populate-products - Add sample products

üîß Troubleshooting

  • MongoDB Connection Issues: Make sure MongoDB is running and the connection string in .env is correct
  • Port Already in Use: Change the PORT value in your .env file
  • Email Issues: Verify your email credentials and enable "Less secure app access" for Gmail
  • Permission Errors: Run commands with appropriate permissions or use sudo on Unix systems
